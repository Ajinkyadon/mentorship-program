## What's Next After This Program

### Path 1: Job Search (You're Ready)

```
Month 1: Applications
  → 20+ applications per week
  → LinkedIn + Naukri + AngelList + company websites
  → Referrals (reach out to LinkedIn connections)
  → Off-campus drives

Month 2: Interview rounds
  → Aptitude online tests
  → Technical rounds
  → HR rounds
  → Negotiate offer (yes, freshers can negotiate too)

Month 3: Offer + Joining
  → Notice, joining formalities
  → Keep learning even after offer
```

### Path 2: Strengthen Skills (More Preparation)

Agar 2–3 areas mein rating 2 se kam hai:
```
4–6 more weeks on weak areas
→ More DSA practice
→ More projects
→ More mock interviews
Then back to job search
```

```
Upwork, Toptal, Freelancer — start with small projects

Portfolio sites — even small businesses need them
```

**Year 1–2:**
- Architecture decisions participate karo
- Mentor junior team members

- Tech blogs likhna shuru karo
