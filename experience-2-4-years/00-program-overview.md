## Program Vision

> **"Sirf framework use karna band karo — systems banana seekho."**

Most 2–4 year developers can write code. Few can:
- Design scalable architecture
- Debug production issues
- Handle 1M+ users
- Talk confidently in system design rounds
- Own an entire feature end-to-end

This program bridges that gap.

### 5 Pillars

```
1. DEPTH        → Surface-level nahi — internals samajhna
2. SYSTEMS      → Ek API nahi — poora scalable system banana
3. PRODUCTION   → Local nahi — real cloud deployment
4. INTERVIEWS   → Fresher level nahi — senior-level rounds crack karna
5. OWNERSHIP    → "Main karunga" mentality develop karna
```

## Weekend Learning Plan (For Working Professionals)

**Weekdays (1.5 hours/day):**
```
Mon: Concept study (30 min) + implement small piece (1 hr)
Tue: Continue implementation
Wed: Review + fix gaps + refactor
Thu: Add tests + documentation
Fri: Code review your own PR + plan weekend
```

**Weekend (4–6 hours total):**
```
Sat Morning (2 hrs): Deep concept — system design, architecture
Sat Evening (1 hr): LeetCode medium (2 problems)
Sun Morning (2 hrs): Project work — build a feature
Sun Evening (1 hr): Review week's learning, plan next week
```

## FAQs

**Q: 2 saal experience hai, kya ye program mere liye hai?**
A: Haan. Program 2–4 years ke liye designed hai. Agar basics solid hain toh start kar sakte ho.

**Q: Kitne ghante daily dene padte hain?**
A: Minimum 1.5 ghante weekdays, 4-5 ghante weekend. Working professionals ke liye ye feasible hai.

**Q: Kya Node.js mandatory hai?**
A: Examples Node.js mein hain but concepts language-agnostic hain. Java/Python developers bhi benefit karenge.

**Q: Product company switch possible hai 6 months mein?**
A: Possible hai — agar consistently follow karo. Results effort pe depend karte hain.

**Q: System design bilkul nahi aata — kya ye program help karega?**
A: Haan, system design ko scratch se build karte hain — structured approach se.

**Q: DSA kitna focus hai?**
A: Working professionals ke liye optimized — 2–3 months dedicated practice with patterns, not grinding.

**Q: Kya cloud experience chahiye pehle se?**
A: Nahi. AWS section scratch se start hota hai — beginner friendly.

**Q: Salary negotiation bhi cover hoti hai?**
A: Haan — Module 14 mein complete career growth section hai including negotiation strategies.
