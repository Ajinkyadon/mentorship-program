# EngineerUp — Mid-Level to Senior Developer Acceleration Program
### "Framework Developer se Senior Engineer tak" 🚀

Ye program specially designed hai **2–4 years experience wale Indian software engineers** ke liye jo:
- Service companies se product companies mein switch karna chahte hain
- 15–40 LPA roles target kar rahe hain
- Backend engineering mein depth build karna chahte hain
- System design + production engineering seekhna chahte hain

---

## Who Is This For?

✅ Node.js / backend developers with 2–4 years experience  
✅ Service company developers (TCS, Infosys, Wipro, Cognizant, etc.)  
✅ Developers stuck in support/maintenance projects  
✅ Engineers targeting product companies and startups  
✅ Developers lacking cloud + system design exposure
