# From Support to Developer — Full Stack Mentorship Program

> **A 3-Month Career Transformation Program for Working Professionals**

---

## 🎯 Who Is This For?

This program is built for working professionals in MNCs — tech support engineers, QA testers, IT helpdesk executives, and non-development roles — who have the ambition to become software developers but don't know where to start.

You already have:
- Basic HTML, CSS, JavaScript knowledge
- A professional work ethic
- Real-world problem-solving experience

What you'll gain in 3 months:
- Full-stack development skills (React + Node.js + MongoDB/SQL)
- Production-level project experience
- Interview confidence for **10–15 LPA roles**
- A portfolio that speaks louder than any certificate

## 🗺️ Program Overview

```
MONTH 1 — Foundation & Frontend
├── Week 1-2: JavaScript Deep Dive
├── Week 3-4: React.js Fundamentals
└── Week 5-6: React Advanced + Next.js Intro

MONTH 2 — Backend & Full Stack
├── Week 7-8: Node.js + Express.js
├── Week 9-10: MongoDB + SQL + REST APIs
└── Week 11-12: Authentication, Full Stack Integration

MONTH 3 — Production & Interview Ready
├── Week 13-14: Deployment, Docker, CI/CD
├── Week 15-16: Real Project Building
└── Week 17-18: Mock Interviews + Resume + Job Applications
```

## 🏆 Program Outcome

By the end of 3 months, you will:

- ✅ Build and deploy 3+ full-stack projects
- ✅ Confidently explain your code in interviews

- ✅ Have a professional GitHub portfolio
- ✅ Transition from your current role to a Developer role
